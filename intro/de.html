Was ist die CovAPP?

Die CovAPP ist ein von der Charité entwickeltes Softwareprogramm, mit dem Sie innerhalb weniger Minuten Ihr Risiko für das aktuelle Vorliegen eine Corona-Infektion ermitteln können und hilfreiche Hinweise erhalten. Dafür stellt Ihnen das Programm eine Reihe von Fragen und erstellt Ihnen daraus eine Risikobewertung und weist auf entsprechende Kontaktstellen hin. Über Links gelangen Sie direkt zu ausführlichen Informationen der Charité und des Robert-Koch-Instituts im Hinblick auf Hygiene und Infektionsschutz.

Auf diese Weise kann Ihnen der Symptom-Checker helfen, Ihre gesundheitlichen Probleme besser einschätzen und die für ein Arztgespräch relevanten medizinischen Informationen zusammen zu fassen.

Anamnesedaten zum Mitnehmen

Zum Abschluss der Befragung erhalten Sie einen QR-Code, der später im Fall eines Arztgesprächs in der Corona-Ambulanz der Charité Campus Virchow Klinikum eingelesen werden kann. Bitte bringen Sie zu Ihrem Ambulanzbesuch Ihr Smartphone mit dem QR-Code oder den ausgedruckten Code mit. In der Ambulanz wird der Code gescannt, so dass Ihre Antworten auf den Fragebogen direkt unserem medizinischen Personal für das Arztgespräch zur Verfügung gestellt werden.

Für die Besuche in einer anderen Corona-Ambulanz außerhalb der Charité laden Sie bitte das zur Verfügung gestellte PDF-Dokument herunter und drucken Sie es für Ihr Arztgespräch aus.

Im Zweifel zum Arzt

Die Nutzung dieser App ersetzt keine ärztliche Behandlung. Wenn Sie sich aktuell schwer krank fühlen, suchen Sie bitte umgehend medizinische Hilfe auf.
